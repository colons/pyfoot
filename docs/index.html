pyfoot

A useless, poorly documented* and badly written threaded IRC robot, written in Python. Source available from Bitbucket.

ddg

  • !ddg
  • query

Issue a DuckDuckGo query.

!ddg 2^10

calc : 2 ^ 10 = 1,024 | http://ddg.gg/?q=2%5E10

Supports DuckDuckGo's !bang syntax.

mal

  • !mal
  • user
  • set user
  • fight user user
  • compare user user

A collection of MyAnimeList-related functions.

Most of these functions benefit from knowing who you (and everyone else) are.

!mal set colons

nivi is MAL user colons

You can now get a summary of yourself with !mal, or you can summarise another person by specifying a target.

!mal brie

brie | 0.43 days across 1 shows | Berserk : 25/25 : 10

You can also compare yourself to other users or any two users to each other with !mal fight or !mal compare.

!mal fight xinil

colons vs. xinil | average contention : 1.12 | Tengen Toppa Gurren Lagann : 4 vs. 8

rss

pyfoot is capable of reading an RSS feed into a channel, but it's not configurable live. Talk to whoever is operating your local pyfoot instance if you have a feed you'd like in a channel.

say

  • !say
  • destination phrase

Utterly indiscriminate messaging. Will work in queries, so can be used for anonymisation (although your local pyfoot operator may be able to determine and reveal your identity).

!say #unix I'd just like to interject for a moment.

I'd just like to interject for a moment.

translate

  • !t
  • from to phrase

Translates from from to to using the Bing translate API. from and to must be any of the language codes here and here, respectively.

!t fr en le jambon est mort

the ham is dead
