mal

  • !mal
  • user
  • set user
  • fight user user
  • compare user user

A collection of MyAnimeList-related functions.

Most of these functions benefit from knowing who you (and everyone else) are.

!mal set colons

nivi is MAL user colons

You can now get a summary of yourself with !mal, or you can summarise another person by specifying a target.

!mal brie

brie | 0.43 days across 1 shows | Berserk : 25/25 : 10

You can also compare yourself to other users or any two users to each other with !mal fight or !mal compare.

!mal fight xinil

colons vs. xinil | average contention : 1.12 | Tengen Toppa Gurren Lagann : 4 vs. 8

If desired, you can specify the language to be translated to and from by using !partyvia/!pv lang phrase. lang must be a language code from both here and here, as it will be used repeatedly as both a source and target language.

  • destination phrase

Behaviour

On startup, pyfoot will join all the channels his configuration suggests he should. Channels joined and left while running do not change this.

While running, he will join any channel he recieves an invitation to. He may not be welcome, however, and there's no mechanism on his end for detaching from a channel.

pyfoot will, under no circumstances, automatically rejoin a channel when kicked. If you want him back, re-invite him.
